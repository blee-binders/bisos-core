** Broadly speaking, PyCS files are of of the following:
*** +
*** CSXU  -- main for CS Multi-Units -- Could be Packaged or Standalone. Could be Direct, RO, or Planted
*** CSU   -- A CS Unit -- Always Packaged. -- Could be Generic or Specific
*** PyCS Library  -- Includes No Cmnds and no Params. Classes and Functions and Operations
*** Uploadable Modules -- Used by uploadAsCs
*** Seeded (Plant+Seed)
*** External Files
*** -

Here it is in tabular form:

|---------+---------------+---------+----------+---------+-------------+------------------------------|
| General | Specific      | Pkging  | Summary  | snippet | Naming      | Comments                     |
| Type    | Type          |         | Tag      |         |             |                              |
|---------+---------------+---------+----------+---------+-------------+------------------------------|
| CSXU    |               |         |          |         | xx.cs       | CS Multi-Units with main     |
| XU      |               |         |          |         | xx.cs       | XU (eXecution Unit) aka CSXU |
|---------+---------------+---------+----------+---------+-------------+------------------------------|
|         | [[Pkg-Direct]]    | pip     | CmndSvc+ |         | Pkg, Direct | Examples only, no cmnds      |
|         | [[Single-Direct]] | stndaln | CmndSvc+ |         | Single, Dir | params + cmnds + examples    |
|         | [[Pkg-RO]]        | pip     | CmndSvc+ |         | Pkg, RO     |                              |
|         | plntd-tmplt   | pip     | Planted+ |         | Template    | Planted include with seed    |
|         | plntd-niched  | realmed | Planted+ |         | Niched      | Planted based on realms      |
|         | plntd-spread  | spread  | Planted+ |         | Spread      | Placed in dir structure      |
|---------+---------------+---------+----------+---------+-------------+------------------------------|
| CSU     |               |         |          |         | xx_csu.py   | CS Units, Params + Cmnds     |
|---------+---------------+---------+----------+---------+-------------+------------------------------|
|         | Specific      | pip     | Cs-Unit+ |         | Specific    | Domain Specific Params+Cmnds |
|         | Generic       | pip     | Cs-Unit+ |         | Generic     | General purpose Params+Cmnds |
|---------+---------------+---------+----------+---------+-------------+------------------------------|
| CsLib   |               |         |          |         |             | PyCs-Lib No Params, no Cmnds |
|---------+---------------+---------+----------+---------+-------------+------------------------------|
|         | PyCS-Lib      |         | PyCs-Lib |         |             | Classes, Functions and OPs   |
|         | seed          |         |          |         | xx_seed.py  |                              |
